# 云服务提供商全球节点地图

这是一个展示阿里云、华为云、腾讯云全球节点布局的交互式地图。

## 🌍 功能特性
- 全球节点分布地图
- 实时统计信息
- 按云服务提供商筛选
- 响应式设计
- 交互式地图操作

## 🛠️ 技术栈
- HTML5 + CSS3 + JavaScript
- Leaflet.js 地图库
- OpenStreetMap 地图数据

## 📊 数据来源
- 阿里云官方API
- 华为云官方API
- 腾讯云官方API

## 🔄 更新频率
数据基于官方API实时获取，定期更新。
